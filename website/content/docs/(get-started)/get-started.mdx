Get Started covers the commands you usually run once per machine.

## Flow

```bash
brew install samzong/tap/gmc
gmc init
gmc config get
gmc completion zsh > ~/.zsh/completions/_gmc
```

## What belongs here

- Installation
- Interactive setup
- LLM configuration
- Shell completion
- Bundled Agent Skill installation

After this, use `gmc wt` for parallel worktrees and `gmc` for AI commit messages.
