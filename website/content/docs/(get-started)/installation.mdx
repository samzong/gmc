---
title: Installation
description: Install gmc with Homebrew or Go.
---

Install `gmc` before setting up config or worktree automation.

## Homebrew

```bash
brew install samzong/tap/gmc
```

## Go

```bash
go install github.com/samzong/gmc@latest
```

## Verify

```bash
gmc version
gmc --help
gmc wt --help
```

## Next

Run the setup wizard:

```bash
gmc init
```
