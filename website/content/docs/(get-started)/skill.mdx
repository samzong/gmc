---

`gmc skill` manages the bundled Agent Skill that teaches agents how to use `gmc`.

## Usage

```bash
gmc skill install
```

## When to use it

Run this when an agent environment needs local instructions for worktree commands, commit generation, and common guardrails.

## Notes

The Skill is separate from shell completion. Completion helps your shell; the Skill helps coding agents call `gmc` correctly.
