---
title: Init
description: Initialize gmc configuration.
---

`gmc init` starts the interactive setup wizard.

## Usage

```bash
gmc init
```

## When to use it

Use it on a new machine or when you want to rebuild the main config without editing YAML by hand.

The wizard writes the same values that `gmc config set ...` manages later.

## Related

- `gmc config get`
- `gmc config set model`
- `gmc config set apikey`
