Troubleshooting covers common local setup, API, and worktree cleanup failures.

## Common pages

- Shell alias conflict
- Config issues
- LLM/API failures
- Worktree cleanup

## First checks

```bash
gmc version
gmc config get
gmc --help
```

If a command behaves unlike the docs, confirm that your shell is invoking the `gmc` binary you installed.
