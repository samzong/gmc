---
title: Attach
description: Attach to a task agent session.
---

`gmc task attach` attaches to the agent session for a task.

## Usage

```bash
gmc task attach <task-id>
```

## When to use it

Use it when an agent is already running and you want to inspect or continue the session.

## Notes

Attach depends on the session metadata created by `gmc task start`.
