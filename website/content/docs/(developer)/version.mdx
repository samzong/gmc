---
title: Version
description: Show gmc version information.
---

`gmc version` prints the installed binary version.

## Usage

```bash
gmc version
gmc --version
```

## When to use it

- Verify installation.
- Check which binary is on `PATH`.
- Confirm a release after installing from Homebrew or `go install`.

## Notes

If the output does not match the release you expected, check shell aliases and `PATH` order.
